## Установка и запуск
#### Для установки и запуска проекта необходимо:
Клонировать репозиторий

```
git clone git@github.com:Timur-Abdullaev-07/stellar-burgers.git
```
Установить зависимости
```
npm install
```
Скопировать .env.example в .env
```
cp .env.example .env
```
Запустить проект
```
npm run start
```
## Сборка

```
npm run build
```
